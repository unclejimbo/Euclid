Detailed Description

This package provides functions to make CGAL mesh from raw buffers and vice versa.

Conceptually, the CGAL mesh type should conform to the concepts defined in the Boost Graph Library, including CGAL::Surface_mesh, CGAL::Polyhedron_3 and OpenMesh. For more details, see https://doc.cgal.org/latest/BGL/index.html.
